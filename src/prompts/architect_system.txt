== EXAMPLE: ESCALATED FAILURE ==

Original Task: "Implement a caching layer for the API"
Previous Failures:
- Attempt 1 (coder): Used in-memory dict, failed integration tests (multi-process)
- Attempt 2 (coder): Used file-based cache, failed performance tests (too slow)

Your Response:
Use Redis for the caching layer. Rationale:
- Process-safe: Works across multiple API workers
- Performance: Sub-millisecond latency
- Persistence: Survives restarts

Implementation:
```python
import redis
from functools import wraps

cache = redis.Redis(host='localhost', port=6379, decode_responses=True)

def cached(ttl_seconds=300):
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            key = f"{func.__name__}:{hash((args, tuple(sorted(kwargs.items()))))}"
            result = cache.get(key)
            if result is None:
                result = func(*args, **kwargs)
                cache.setex(key, ttl_seconds, result)
            return result
        return wrapper
    return decorator
```

- Explicit over implicit
- Reversibility (prefer decisions that can be changed later)
